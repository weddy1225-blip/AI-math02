# 🕵️ 數字雙胞胎找一找

這是一款專為 **國小四年級學生** 設計的 AI 互動數學邏輯遊戲。透過觀察與推理，訓練學生的數感、四則運算邏輯以及初步的代數等價概念。

## 🎯 遊戲設計目標
* **強化數感**：透過「？」挖空算式，讓學生練習逆向運算。
* **邏輯判斷**：不只是算出答案，更要觀察兩邊算式的結構是否對等（例如：加數與和的變化關係）。
* **無壓力學習**：自動化出題與即時回饋，適合學生在家自主利用電腦、手機或平板操作。

## 🎮 遊戲規則
1. 每場挑戰共有 **10 道關卡**。
2. 畫面會同時出現左右兩個算式，每個算式都有一個「？」挖空。
3. 偵探任務：判斷左右兩邊的「？」是不是代表 **同一個數字**。
4. 點選「一樣」或「不一樣」進行破案。
5. 完成 10 題後，系統將根據準確度給予偵探評分（總分 100 分）。

## 📝 題型涵蓋範圍
針對小四數學課程調整，隨機產生以下四類案件：
* **純加法案件**：基礎數值位移觀察。
* **純減法案件**：練習被減數與差的關係。
* **加減混合案**：進階逆向思考。
* **四則運算挑戰**：加入基礎乘除法的邏輯對照。

## 🚀 快速開始 (部署方式)
本專案採用純前端技術開發，部署極為簡單：
1. 將本專案的 `index.html` 上傳至 GitHub 儲存庫。
2. 進入 **Settings > Pages**。
3. 在 **Branch** 選擇 `main` 並點擊 **Save**。
